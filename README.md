# Bank Product Profitability Analysis

Analyze profitability, marketing ROI, and customer churn of banking products using **SQL, Power BI, and Python**.

---

### Overview
Built an end-to-end analytics workflow for bank products (Loans, Deposits, Credit Cards, Insurance):
- Cleaned and joined data in **MySQL**
- Computed KPIs: Profit Margin, ROI, Churn %, NIM
- Forecasted revenue & churn using **Prophet / SARIMAX**
- Visualized results in an interactive **Power BI dashboard**

---

### Tech Stack
**SQL**,  **MySQL**

---

### Key Insights
- Loans & credit cards gave high profits but higher churn  
- Deposits achieved best marketing ROI  
- Net Interest Margin stable around 2.4%  
- Forecasts show steady revenue growth and lower churn

---

### How to Run
1. Execute SQL scripts → create tables & views  
2. Export data to CSV → analyze in Python notebooks  
3. Load into Power BI → view dashboard KPIs
